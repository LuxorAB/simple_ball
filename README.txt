# simple_ball
Предварительное описание концепции

•	Жанры: «платформер», «головоломка», «сайд-скроллер».
•	Предполагаемый движок: Unity (бесплатная лицензия https://unity3d.com) 
•	Основная особенность: эмуляция физических свойств сред и объектов

Цель игрока, провести шар до цели по уровню, изменяя физические параметры шара, окружающей среды и других объектов.
Основной режим управления – вращение шара по и против часовой стрелки.
Вероятны альтернативные режимы перемещения шара, такие как толчки в заданном направлении, изменение вектора гравитации.

Используемые параметры (предварительный список)
Для шара	
  Масса	m
	Плотность	ρ = m/V
	Объем	V
	Трение (скольжения)	F = kN
	Упругость	
	Липкость	
	Температура	t
Для объектов окружения, элементов локации
  Трение (скольжения)	F = kN
	Упругость	
	Липкость	
	Температура	t
Для сред
  Плотность (жидкости, газы)	ρ = m/V
	Температура	t
	Ускорение свободного падения (гравитация)	g

Список используемых переменных
•	m – масса
•	ρ – плотность
•	V – объем
•	F – трение
•	k – коэффициент трения скольжения
•	N – Сила нормальной реакции опоры https://ru.wikipedia.org/wiki/Сила_нормальной_реакции
•	t – температура
•	g – ускорение свободного падения
